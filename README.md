# animatedgifs_app
음성과 얼굴 합성을 통해 영화/드라마 속 명장면의 주인공이 되어보는 어플.



기능
----
1. 영화/드라마 명장면 리스트
2. 회원가입
3. 얼굴 인식
4. 음성 녹음
5. 얼굴 합성
6. 동영상 다운로드



사용기술
--------
● 서버: GCP(Google Cloud Platform)

● 웹 서버: Nginx, uwsgi, flask

● 데이터베이스: Mysql

● 언어: Java, Python

● 라이브러리/API : OpenCV, Face Landmark, FFmpeg
